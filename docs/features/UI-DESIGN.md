## 二、设计风格

### 2.1 视觉风格

| 属性 | 值 |
|-----|-----|
| 主题 | 暗色主题 (Dark Theme) |
| 主色调 | #4A90E2 (蓝色) |
| 成功色 | #4CAF50 (绿色) |
| 警告色 | #FF9800 (橙色) |
| 危险色 | #E53935 (红色) |
| 字体 | Noto Sans SC (中文) + JetBrains Mono (代码) |

### 2.2 设计特点

- **玻璃态效果**：半透明背景 + 背景模糊
- **渐变卡片**：四种主题色渐变
- **悬浮交互**：卡片悬浮抬升效果
- **状态指示器**：带发光效果的状态点
- **平滑动画**：fadeIn、pulse、slideIn

## 四、组件设计

### 4.1 导航栏组件

```html
<nav class="nav-item">
    <span>图标</span>
    <span>菜单文字</span>
</nav>
```

**状态**：
- 默认：灰色文字
- 悬浮：浅蓝背景 + 蓝色左侧边条
- 激活：浅蓝背景 + 蓝色左侧边条 + 白色文字

### 4.2 统计卡片

```html
<div class="gradient-{color} rounded-2xl p-5">
    <div class="text-4xl font-bold">数值</div>
    <div class="text-sm">标题</div>
    <div class="text-xs mt-2">趋势</div>
</div>
```

**渐变类型**：
- `gradient-blue`：蓝色渐变
- `gradient-green`：绿色渐变
- `gradient-orange`：橙色渐变
- `gradient-purple`：紫色渐变

### 4.3 账号卡片

```html
<div class="bg-[#252526] rounded-2xl p-5 card-hover">
    <div class="platform-icon bg-{color}">平台字</div>
    <div class="status-dot status-{state}"></div>
    <h4>账号名称</h4>
    <p>@username</p>
</div>
```

**平台颜色**：
- 知乎：#0084FF (蓝色)
- 百家号：#E53935 (红色)
- 搜狐号：#FF6B00 (橙色)
- 头条号：#333333 (黑色)
- 公众号：#07C160 (绿色)

### 4.4 发布流程步骤

```
┌───┐    ┌───┐    ┌───┐    ┌───┐
│ 1 │────│ 2 │────│ 3 │────│ 4 │
└───┘    └───┘    └───┘    └───┘
激活     激活     禁用     禁用
```

**步骤**：
1. 选择要发布的文章
2. 选择发布账号
3. 确认发布
4. 查看进度

## 五、交互设计

### 5.1 页面切换

```javascript
// 导航点击切换页面
navItems.forEach(item => {
    item.addEventListener('click', () => {
        // 更新导航状态
        // 切换页面内容
        // 更新标题
    });
});
```

### 5.2 复选框

```javascript
// 自定义复选框交互
checkbox.addEventListener('click', () => {
    checkbox.classList.toggle('checked');
});
```

### 5.3 动画效果

| 动画 | 用途 | 时长 |
|-----|------|------|
| fadeIn | 页面内容进入 | 0.3s |
| pulse | 加载中状态 | 2s 循环 |
| slideIn | 侧边栏进入 | 0.3s |

## 六、响应式设计

### 6.1 断点

| 尺寸 | 说明 |
|-----|------|
| < 768px | 移动端（暂不支持） |
| 768px - 1024px | 平板（侧边栏可折叠） |
| > 1024px | 桌面端（完整布局） |

### 6.2 卡片网格

- 桌面端：3列网格
- 小屏：2列或1列自适应
